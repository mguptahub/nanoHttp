Installation

Quick Install

curl -fsSL https://justhttp.mguptahub.com/install.sh | bash

Manual Install

# Download the binary curl -LO https://github.com/mguptahub/justHttp/releases/latest/download/justHttp-linux-amd64 # Make it executable chmod +x justHttp-linux-amd64 # Move to /usr/local/bin sudo mv justHttp-linux-amd64 /usr/local/bin/justHttp
# Download the binary curl -LO https://github.com/mguptahub/justHttp/releases/latest/download/justHttp-darwin-amd64 # Make it executable chmod +x justHttp-darwin-amd64 # Move to /usr/local/bin sudo mv justHttp-darwin-amd64 /usr/local/bin/justHttp

Usage

Add a Server Instance

# Add a server on port 8080 justHttp add --port 8080 # Add a server with custom config justHttp add --port 8081 --config custom.yaml

Manage Instances

# List all instances justHttp list # Start all instances justHttp start all # Stop a specific instance justHttp stop 8080 # Remove an instance justHttp remove 8080
